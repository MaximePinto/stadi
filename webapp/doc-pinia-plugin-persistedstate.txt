TITLE: Pinia Persisted State Plugin Setup Examples
DESCRIPTION: Demonstrates how to enable and configure state persistence for Pinia stores using both the setup and options API syntaxes. It shows where to place the `persist` configuration object.

SOURCE: https://github.com/prazdevs/pinia-plugin-persistedstate/blob/main/docs/guide/config.md#_snippet_0

LANGUAGE: ts
CODE:
```
import { defineStore } from 'pinia'
import { ref } from 'vue'

export const useStore = defineStore('main', () => {
  const someState = ref('hello pinia')
  return { someState }
}, {
  persist: {
    // CONFIG OPTIONS HERE
  }
})
```

LANGUAGE: ts
CODE:
```
import { defineStore } from 'pinia'

export const useStore = defineStore('main', {
  state: () => ({
    someState: 'hello pinia',
  }),
  persist: {
    // CONFIG OPTIONS HERE
  },
})
```

TITLE: Enable Persistence (Setup Syntax)
DESCRIPTION: Demonstrates how to enable state persistence for a Pinia store defined using the setup function syntax. Set the `persist` option to `true` within the store's configuration.

SOURCE: https://github.com/prazdevs/pinia-plugin-persistedstate/blob/main/docs/guide/index.md#_snippet_4

LANGUAGE: ts
CODE:
```
import { defineStore } from 'pinia'
import { ref } from 'vue'

export const useStore = defineStore(
  'main',
  () => {
    const someState = ref('hello pinia')
    return { someState }
  },
  {
    persist: true,
  },
)
```

TITLE: Add Plugin to Pinia Instance
DESCRIPTION: Integrates the pinia-plugin-persistedstate into your Pinia application instance. This step is crucial for enabling the persistence functionality across your stores.

SOURCE: https://github.com/prazdevs/pinia-plugin-persistedstate/blob/main/docs/guide/index.md#_snippet_3

LANGUAGE: ts
CODE:
```
import { createPinia } from 'pinia'
import piniaPluginPersistedstate from 'pinia-plugin-persistedstate'

const pinia = createPinia()
pinia.use(piniaPluginPersistedstate)
```
